Установка и настройка (Быстрый старт)

Для запуска проекта на вашем компьютере должен быть установлен **[Node.js](https://nodejs.org/)**.

1. Установка зависимостей

Откройте терминал в папке с проектом и выполните команду для скачивания необходимых библиотек:
```bash
npm install
```
2. Настройка переменных окружения (.env)

В корневой папке необходимо отредактировать файл `.env`.  Подставьте свои данные от аккаунта Mailtrap для тестирования почты в следующие строки:
```
MAIL_USER= (нужно ввести логин)
MAIL_PASS= (нужно ввести пароль)
```
3. Инициализация базы данных

База данных apexmind.db и таблица с администратором создадутся автоматически при первом запуске сервера. Никаких дополнительных команд для БД вводить не нужно.

ЗАПУСК ПРОЕКТА
1. Запуск Backend-сервера

В терминале (находясь в папке проекта) введите:
```bash
node server.js
```
В консоли должно появиться сообщение: Сервер запущен на http://localhost:5000 (или на том порту, который указан в коде). Сервер должен работать в фоновом режиме всё время, пока вы пользуетесь сайтом.

2. Запуск Frontend (Сайта)
Откройте файл ```index.html``` в браузере.

Панель администратора
Для просмотра статистики опросов и оставленных заявок:
Откройте файл ```admin.html``` в браузере.
Введите данные для входа (создаются по умолчанию):

Логин: ```admin```

Пароль: ```123```
